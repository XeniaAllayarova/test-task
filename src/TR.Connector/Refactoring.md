# Рефакторинг Connector

Проведен рефакторинг Connector. Изменения:

1. Выделение ApiClient, что устранило дублирование http-запросов и инициализации httpClient в каждом методе.
2. Добавление обработки ошибок в http-запросах
3. Выделение общих методов, в том числе добавлены методы расширения
4. Использование констант
5. Улучшение работы с данными
6. Добавление структуры проекта
